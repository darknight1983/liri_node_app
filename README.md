

# LIRI_NODE_APP


**Install these npm packages.**

* npm install twitter
* npm install node-spotify-api
* npm install request-promise
* npm install dotenv


**Commands to run LIRI:**

* my-tweets

* spotify-this-song <your-song-goes-here>

* movie-this

* do-what-it-says
